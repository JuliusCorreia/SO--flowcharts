Interactive Security Flowcharts

Tap a flow, then tap each IF step to reveal the THEN outcome.

1 Overall Passenger Screening (People) ▶
IF passenger arrives at lane
▶
  • THEN greet the passenger.
  • THEN explain the screening process.
  • THEN direct them to the divest/loading area.
IF passenger has removed required items (coats, belts, shoes, pockets empty)
▶
  • THEN send the passenger to the QPS entry point.
IF passenger is an exception (wheelchair, infant < 1 m, pacemaker/medical, very tall > 2 m)
▶
  • THEN skip QPS.
  • THEN go to adapted search (manual search + HHMD + ETD as required).
IF passenger is not an exception
▶
  • THEN send the passenger through QPS once.
IF QPS is clear (no activations)
▶
  • THEN the passenger goes to collect bags.
  • THEN the passenger exits the screening area.
IF QPS shows any activation
▶
  • THEN follow the QPS Decision / Activation Flows (2 & 3).
2 QPS Decision Flow (Use / Skip) ▶
IF passenger can stand correctly, is ≥ 1 m tall, and has no QPS medical exemption
▶
  • THEN use QPS for screening.
IF passenger is < 1 m tall
▶
  • THEN do not use QPS.
  • THEN go to adapted manual search + HHMD/ETD as appropriate.
IF passenger has medical reason OR cannot assume correct pose
▶
  • THEN do not use QPS.
  • THEN call SM.
  • THEN perform adapted search (manual + HHMD/ETD as per SOP).
IF passenger is > 2 m tall (zone 2 above QPS limit)
▶
  • THEN call SM.
  • THEN perform adapted search (partial QPS plus manual/HHMD if allowed).
IF passenger refuses QPS for non-medical reasons
▶
  • THEN call SM.
  • THEN move to private search.
  • THEN remove shoes for X-ray.
  • THEN conduct full physical search + HHMD + ETD.
  • THEN ensure any rejected bags remain in the search lane.
3 QPS Result / Activation Flow ▶
IF QPS result = “clear” (no activations)
▶
  • THEN passenger proceeds to collect bags and leave.
IF QPS shows single or multiple zones activated (not RND only)
▶
  • THEN direct passenger to same-gender searcher.
  • THEN use HHMD on activated zones.
  • THEN follow HHMD flow to resolve activations.
IF QPS shows RND at left ankle with no real activations
▶
  • THEN send passenger to SAMDEX (shoe scanner).
  • THEN follow SAMDEX flow.
IF QPS shows RND plus real activations in other zones
▶
  • THEN treat non-RND zones as real activations.
  • THEN use HHMD/physical search on those zones.
  • THEN also send passenger to SAMDEX if RND indicates shoes.
IF QPS result = “inadequate scan”
▶
  • THEN correct passenger posture.
  • THEN repeat the scan once.
IF QPS is “inadequate” twice in a row
▶
  • THEN stop re-scanning.
  • THEN perform a full physical body search instead.
IF QPS result = “insufficient quality” (system issue)
▶
  • THEN ask passenger to step back from the scanner.
  • THEN call SM.
  • THEN SM decides next step (manual search or re-routing).
4 SAMDEX (Shoe Scanner) Flow ▶
IF QPS result has RND with shoes on
▶
  • THEN send passenger to SAMDEX.
IF QPS result has zone 4 activation
▶
  • THEN send passenger to SAMDEX (unless they wear boots above the ankle).
IF passenger is wearing boots above the ankle
▶
  • THEN do NOT use SAMDEX.
  • THEN remove boots and send them through X-ray instead.
IF SAMDEX result = green
▶
  • THEN shoes are cleared; no extra shoe action required.
IF SAMDEX result = yellow (first time)
▶
  • THEN repeat the test on SAMDEX.
IF SAMDEX result = yellow twice in a row
▶
  • THEN treat as real activation.
  • THEN escalate to SM.
  • THEN remove shoes.
  • THEN perform zone 4 physical search + HHMD.
  • THEN send shoes through X-ray.
IF SAMDEX result = blue (metal) OR orange (possible explosive)
▶
  • THEN escalate to SM.
  • THEN remove shoes.
  • THEN perform zone 4 physical search + HHMD.
  • THEN send shoes through X-ray.
IF medical condition prevents correct SAMDEX stance
▶
  • THEN remove shoes and X-ray them.
  • IF shoes cannot be removed → ETD shoes and fastenings + physical check on shoes.
  • No private search required solely for this.
IF passenger refuses SAMDEX (non-medical)
▶
  • THEN call SM.
  • THEN move to private search.
  • THEN shoes removed and X-rayed.
  • THEN full physical search + HHMD + ETD.
  • THEN any rejected bags remain in the search lane.
5 HHMD (Hand-Held Metal Detector) Flow ▶
IF physical body search is required AND no HHMD restriction
▶
  • THEN use HHMD on relevant body areas.
IF HHMD does NOT activate on an area
▶
  • THEN proceed to next area or finish search when complete.
IF HHMD activates on an area
▶
  • THEN search a 6-inch circle around that point with HHMD.
  • THEN conduct physical touch search in that area.
IF HHMD activates but no prohibited item found and suspicion remains
▶
  • THEN consider ETD swabs on that area or clothing.
  • THEN call SM if concern continues.
IF passenger has pacemaker or device where HHMD is contraindicated
▶
  • THEN do not use HHMD on that area.
  • THEN use adapted manual search plus ETD as appropriate.
6 ETD on People Flow ▶
IF random selection / system activation / profiling requires ETD on passenger
▶
  • THEN choose ETD areas: fastenings, palms, shoes, waistband, headwear.
IF passenger has buttons or fastenings
▶
  • THEN swab buttons and both sides of both palms.
IF passenger has shoes on
▶
  • THEN swab across shoe surfaces, around the back and along laces.
IF passenger has no shoes
▶
  • THEN swab the waistband area instead.
IF passenger wears religious headwear
▶
  • THEN use “halo and mohawk” pattern when swabbing headwear.
IF ETD result = green
▶
  • THEN no explosive trace detected; continue the process as normal.
IF ETD result = red (first time)
▶
  • THEN repeat ETD test using a new sample.
IF ETD result = red twice
▶
  • THEN escalate to SM.
  • THEN follow local escalation and further search/actions.
IF ETD result = yellow
▶
  • THEN refer to SM for decision (re-test, second machine, or escalate).
7 Body / Private Search Flow ▶
IF QPS activation cannot be resolved OR passenger refuses QPS OR strong profiling concern exists
▶
  • THEN arrange a body search.
IF search is in public and passenger accepts public search
▶
  • THEN perform full physical search with HHMD/ETD as required.
IF passenger requests private search OR SM decides private search is needed
▶
  • THEN move passenger to a private room.
IF private search is conducted
▶
  • THEN ensure same-gender officer and witness are present.
  • THEN obtain consent form as required.
  • THEN perform full physical search + full HHMD and ETD checks.
  • THEN X-ray shoes as part of the process.
IF during body search a prohibited but legal item is found
▶
  • THEN inform SM discreetly.
  • THEN complete the search.
  • THEN handle the item as prohibited but legal (remove, offer options).
IF during body search a suspected FA/EX/IID/IED is found
▶
  • THEN stop the search immediately.
  • THEN do NOT touch the item further.
  • THEN discreetly call SM.
  • THEN hold passenger safely and await instructions.
8 Under‑18 / Safeguarding Flow ▶
IF passenger is under 18 years old
▶
  • THEN ensure a parent or guardian is present as witness for any search.
IF no parent/guardian is available
▶
  • THEN use a lane partner as witness (per safeguarding SOP).
IF parent/guardian leaves during search
▶
  • THEN stop the search immediately.
  • THEN wait until the parent/guardian returns.
IF parent/guardian returns
▶
  • THEN explain what will happen.
  • THEN restart or continue search with them present.
IF no suitable adult can stay to witness
▶
  • THEN follow local safeguarding procedure.
  • THEN consult SM before proceeding further.
9 Overall Tray & Bag Flow ▶
IF passenger arrives at loading area
▶
  • THEN instruct them to remove coats, belts, shoes, LAGs and large metal items.
  • THEN ensure items are placed into trays.
IF trays are ready and tidy (1 bag per tray, items flat)
▶
  • THEN send trays through X‑ray/CT.
IF image is clear (no alarms, no suspicion)
▶
  • THEN mark tray as cleared.
  • THEN passenger collects bag and leaves screening area.
IF there is an EDS alarm, shield/density alarm, or operator suspicion
▶
  • THEN reject tray to bag search area for manual resolution.
10 X‑ray / CT Decision Flow ▶
IF CT/X‑ray image shows no anomalies AND no alarms
▶
  • THEN mark bag clear.
IF image shows dense area with shield/density alarm
▶
  • THEN reject bag for manual search.
IF EDS draws a red box (possible explosive)
▶
  • THEN reject bag for search and treat as higher priority.
IF operator sees obvious firearm or IED shape
▶
  • THEN press Obvious Threat button.
  • THEN reject bag and escalate to SM/response.
IF operator is unsure but sees suspicious item
▶
  • THEN use zoom/rotate/slice tools.
  • IF still concerned → reject bag for search and note concern.
11 Bag Search Flow ▶
IF bag is rejected from X‑ray/CT
▶
  • THEN move bag to search area.
IF passenger gives consent (“Yes, you may search my bag”)
▶
  • THEN begin the bag search.
IF passenger refuses search
▶
  • THEN call SM.
  • THEN follow refusal SOP (possible refusal to carry).
DURING search – IF item appears harmless and matches image
▶
  • THEN move that item to the clear tray.
DURING search – IF item is of concern (dense, concealed, unusual)
▶
  • THEN test item with ETD or EMA‑4, or inspect more thoroughly.
IF prohibited but legal item is found (e.g. large scissors, tools, excess liquids)
▶
  • THEN remove item and place it out of passenger’s reach.
  • THEN complete the search.
  • THEN explain and offer options: return landside, left luggage, hold baggage (if allowed), or voluntary abandon.
IF prohibited and illegal/dangerous item is found (e.g. certain weapons)
▶
  • THEN remove the item and place it out of reach.
  • THEN complete the search.
  • THEN discreetly call SM.
  • THEN record search as complete and await further instructions.
IF IID, firearm, component, or suspected explosive is found
▶
  • THEN stop search immediately.
  • THEN do not touch the item further.
  • THEN move tray away if safe.
  • THEN ask customer to remain in place.
  • THEN discreetly call SM and confirm search is incomplete.
12 LAGs and EMA‑4 Flow ▶
IF LAG exceeds standard rules OR needs secondary check
▶
  • THEN test with EMA‑4 (if container type is allowed).
IF container is NOT suitable for EMA‑4 (cup, bag of fluid, flask, leaking, refillable metal)
▶
  • THEN follow local LAG rules (may require disposal or alternative handling).
  • THEN do NOT use EMA‑4 on this container.
IF EMA‑4 result = green
▶
  • THEN LAG is cleared.
IF EMA‑4 result = red
▶
  • THEN leave container in cavity.
  • THEN call SM; likely leads to disposal or abandonment decision.
IF EMA‑4 result = yellow (first time)
▶
  • THEN retest on same machine.
IF EMA‑4 result = yellow twice on same machine
▶
  • THEN test on a second EMA‑4 unit.
IF second machine result = green
▶
  • THEN clear the LAG.
IF second machine result = yellow
▶
  • THEN call SM.
  • THEN SM checks procedure, profiles passenger, and may require extra search or disposal.
13 ETD on Bags / Items Flow ▶
IF bag contains wrapped items, electricals, organ containers, or camera film
▶
  • THEN consider ETD swabs on those items as per SOP.
IF ETD result on item = green
▶
  • THEN no explosive trace detected on that item.
IF ETD result = red (first time)
▶
  • THEN repeat ETD on a fresh swab from that item.
IF ETD result = red twice
▶
  • THEN escalate to SM.
  • THEN consider extra search, passenger questioning, or further action.
IF ETD result = yellow
▶
  • THEN refer to SM.
  • THEN follow local procedure (e.g. retest or second machine).
14 Handling Prohibited Items (Bags) ▶
IF item discovered is prohibited but legal (e.g. scissors, tools, oversize liquids)
▶
  • THEN remove item and keep it out of passenger’s reach.
  • THEN complete the bag search.
  • THEN inform passenger and offer options:
    • Return to landside.
    • Place in hold baggage (if allowed).
    • Use left-luggage service.
    • Voluntarily abandon the item.
IF item discovered is prohibited AND illegal or clearly dangerous (e.g. certain weapons)
▶
  • THEN remove item and place out of reach.
  • THEN complete the search.
  • THEN discreetly call SM.
  • THEN follow incident and recording procedures.
IF item appears to be FA/EX/IID/IED
▶
  • THEN stop search immediately.
  • THEN do not touch the item.
  • THEN move tray away if safe.
  • THEN request customer to remain.
  • THEN discreetly call SM and confirm search incomplete.
15 Special Items & Exemptions (Bags) ▶
IF item is in a STEB (duty‑free liquids)
▶
  • THEN check seal and receipt.
  • IF intact and compliant → allow as per rules.
  • IF tampered or no receipt → treat as standard LAG.
IF bag contains cremated remains
▶
  • THEN allow X‑ray.
  • IF image suggests > 100 g or prohibited article → reject and perform visual/physical inspection.
IF human organs are present
▶
  • THEN do NOT X‑ray the container.
  • THEN perform ETD on the container.
  • THEN involve SM immediately.
IF camera film is present
▶
  • THEN escalate to SM.
  • THEN perform hand-check instead of X‑ray where possible.
IF item is an evidence bag
▶
  • THEN allow X‑ray.
  • THEN do NOT open or physically search the bag.
  • THEN follow instructions from SM/police.
IF item is a diplomatic bag or VIP‑sealed bag
▶
  • THEN once cleared by SM, treat as exempt from normal screening.
  • THEN do NOT X‑ray further.
IF item is complex/oversized (e.g. instruments)
▶
  • THEN call SM.
  • THEN perform full hand check (and X‑ray if permitted).
IF item is non‑complex but cannot fit in tray (pushchair, pram, aid)
▶
  • THEN empty it completely.
  • THEN X‑ray if possible.
  • AND/OR pass through QPS + full hand check depending on item size/type.
